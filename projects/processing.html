Processing Typography

  • Back to all projects

In my UXD study, I took part in a programming bootcamp for one week. During this week, I was introduced to programming in Java using Processing.

Every day, we were learning something new about programming in Java. On the final day, we were given a creative assignment in which we were allowed to create whatever we wanted in Processing.

I decided to combine the knowledge I had gained every day and create a typographic play on words. I used a palindrome, which is a word that creates another word when read backwards. The letters would change size according to the mouse position and order when the user would click.

To achieve this, I mostly made use of conditionals, arrays and functions. The bootcamp taught me the basics about Java and from there, I plan on constantly improving!
